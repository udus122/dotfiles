# Semantic Commit

大きな変更を意味のある最小単位に分割して、セマンティックなコミットメッセージと共に順次コミットします。

## オプション

- `--dry-run`: 実際のコミットは行わず、提案されるコミット分割のみを表示
- `--lang <言語>`: コミットメッセージの言語（デフォルト: en）

## 実行

このコマンドは semantic-committer サブエージェントに処理を委譲します。

サブエージェントは semantic-commit スキルを使用して:
1. 現在の変更を分析
2. 論理的な単位でコミット分割案を作成
3. ユーザー確認後、順次コミット実行
4. 完了報告

引数: $ARGUMENTS

semantic-committer エージェントを呼び出して処理を開始してください。
